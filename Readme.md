# QuickBite

QuickBite is a Django-based online food delivery web application that allows users to browse restaurants, order food, manage profiles, and track deliveries.

---

## Features

- User Authentication System
- Restaurant Listing and Menu Management
- Food Ordering and Cart System
- Order Tracking
- Payment Integration
- Rider and Partner Registration
- Responsive Frontend Design
- Profile Management
- Error Handling Pages

---

## Technology Stack

- Python 3
- Django
- HTML5
- CSS3
- JavaScript
- SQLite3
